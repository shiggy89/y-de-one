# SEOコピーライティング

Y-de-ONEのSEOコピーライティングワークフローを実行します。

## このスキルでできること

1. **キーワードリサーチ**（Googleサジェスト）
2. **ペルソナに合わせたキーワード分類**
3. **ページの見出し・リード文・metadataのSEO改善**

## ペルソナ（常に意識すること）

- **ターゲット**: 60代主婦、お金と時間に余裕あり
- **目的**: 体験レッスンを増やす
- **地域**: 新宿・高田馬場・落合・東中野

---

## ワークフロー

### Step 1: キーワードリサーチ

Googleサジェストスクリプトを実行してキーワードを取得します。

```bash
cd /Users/shiggy/Desktop/y-de-one/scripts
python3 keyword_suggest.py
```

スクリプトが存在しない場合は `scripts/keyword_suggest.py` を作成してから実行してください。

### Step 2: キーワード分類

取得したキーワードをペルソナの悩みに合わせて以下のカテゴリに分類してください。

| カテゴリ | 例 |
|---|---|
| 年齢の不安 | 大人バレエ 何歳から |
| 初心者の不安 | 大人バレエ 初心者・難しい |
| 体験・入口 | 大人バレエ 体験レッスン・服装 |
| 費用・料金 | 大人バレエ 料金・費用・初期費用 |
| 通いやすさ | 大人バレエ 週1・新宿・高田馬場 |
| 健康・効果 | 大人バレエ 効果・ダイエット・痩せる |
| ネガティブ解消 | 大人バレエ みっともない・難しい |

### Step 3: ページ文言のSEOチェック

各ページの見出し・リード文を確認し、以下の観点で評価・改善してください。

**チェック項目**
- [ ] H1にメインキーワード（大人バレエ）が入っているか
- [ ] metadataのtitle・descriptionが設定されているか（デフォルト値のままになっていないか）
- [ ] 各H2がキーワードの羅列になっていないか（自然な日本語か）
- [ ] leadが見出しの言い換えになっていないか（ユーザーへの価値訴求があるか）
- [ ] 同じキーワードが同じページのH2に重複して入っていないか
- [ ] ペルソナの悩み（年齢・費用・通いやすさ）に応えているか

**修正前にかならずBefore/Afterを提示し、承認を得てから修正すること。**

## SEOルール（Y-de-ONE固有）

- メインキーワード: `大人バレエ`・`大人バレエ教室`・`ワイデワン`・`Y-de-ONE`
- 地域キーワード: `新宿`・`高田馬場`・`落合`・`東中野`
- H1とtitleには必ずメインキーワードを入れる
- H2への固有名詞（Y-de-ONE）の繰り返しは避ける（AboutセクションはOK）
- leadはユーザーの共感・悩みから入ること
